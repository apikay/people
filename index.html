People

Remote Users Management

People.js

Install

Include the people.js in your page.


            people.min.js

Usage

Default usage


            // initialize People.
            var people = new People({url: 'http://localhost:3000'});

            // Login button selector.
            var login = document.querySelector(a#login);

            // Show login form.
            login.addEventListener('click', people.showLogin());

            // User login event.
            people.event.on('login', function(e) {
            // User is logged in.
            people.showUser('user-block');
            });

API

People([options])

Initialize People with optional options.

Options:
  • url - The People server base path. Default: http://localhost:3000
  • loginElementID - To bind the login form to this element. Default: body
  • profileElementID - To bind the profile form to this element. Default: body

            var people = new People({url: 'http://localhost:3000'});

.showLogin([options, callback])

Display the login, register or forgot password form.

Options:
  • from - The deafult form to start with. You can use:
    1. login
    2. register
    3. forgot
Callback:

You can use the callback as an on login event callback.


            var people = new People();

            people.showLogin({form: 'register'}, function() {
            // User is logged in
            people.showUser('user-block');
            });

.hideLogin()

Remove the login, register or forgot password form.


            var people = new People();

            people.hideLogin();

.showProfile()

Display the user profile form.


            var people = new People();

            people.showProfile();

.hideProfile()

Remove the user profile form.


            var people = new People();

            people.hideProfile();

.showUser(elementID)

Display a user info: image, name, logout link.

Element ID

The element ID to bind the user block to.


            var people = new People();

            people.showUser('user-block');

.hideUser()

Remove the user block.


            var people = new People();

            people.hideUser();

.event.on(event, callback)

Event emitter.

Events:
  • login - Login
  • register - Register
  • logout - Logout
  • profileupdate - Profile Update
  • passwordupdate - Password Update
  • passwordreset - Password Reset
Callback:

On event callback.


            var people = new People();

            people.event.on('login', function(e) {
            // User is logged in.
            people.showUser('user-block');
            });

.isUser()

Check if user is logged in. Return boolean, true or false.


            var people = new People();

            if(people.isUser()) {
            //
            }

.getUser()

Get the logged in user data. Return user data or false.


            var people = new People();

            var user = people.getUser();

.getInfo()

Get app info. Return app info or false.


            var people = new People();

            var info = people.getInfo();
